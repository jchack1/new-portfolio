Node.js Login/Register App

This app was built using Node.js, Express, and MongoDB Atlas. A user is able to subscribe to a newsletter, register, log in, and add a post to the blog page from their personal dashboard. Register and login functionality was added using Passport.js. Heroku was used for deployment.

The user interface was originally created for an earlier SAIT course about creating static websites. “Undivided mind” is a concept for an online community where artists can share their work, particularly those who wish to harmonize their analytical and creative sides. This concept was a collaborative effort between myself and a friend, who is credited on the site.
