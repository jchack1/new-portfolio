React Scrabble Tile App

This app was built for my Udemy course when I was beginning to learn React. It was created to practice using lists and conditionals and the interface is quite simple. It takes an input from the user and outputs tiles with letters onto the screen, which can be deleted when clicked. The text inside the input field is updated when a deletion is made. I also built a very simple Node.js server in order to host the app on Heroku.
