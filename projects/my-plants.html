My Plant Website

I created this site in order to practice my HTML, CSS, and JavaScript skills. I chose to showcase my plants so I could practice creating galleries using CSS flexbox and grid, and also because the plants are visually appealing and add a pop of colour to the site.

A user can click on the images, toggling text to appear over the image. I created different designs and functionality for desktop, tablet and mobile screen sizes, so please view the site at all sizes to get the full picture of what I can do with basic HTML, CSS, and JavaScript.
